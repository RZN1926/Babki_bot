# 🤖 Telegram-бот для личных финансов

Бот синхронизируется с твоим Firebase — записи мгновенно появляются на сайте.

---

## ⚡ Быстрый старт (Railway — бесплатно)

### Шаг 1 — Создай бота в Telegram
1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Отправь `/newbot`, задай имя и юзернейм
3. Скопируй **токен** (вида `123456789:AAF...`)

### Шаг 2 — Получи сервисный аккаунт Firebase
1. Открой [Firebase Console](https://console.firebase.google.com) → твой проект
2. ⚙️ Настройки проекта → **Сервисные аккаунты**
3. Нажми **"Создать закрытый ключ"** → скачается `serviceAccount.json`
4. Конвертируй в base64:
   ```bash
   # На Mac/Linux:
   base64 -i serviceAccount.json | tr -d '\n'
   # На Windows (PowerShell):
   [Convert]::ToBase64String([IO.File]::ReadAllBytes("serviceAccount.json"))
   ```
5. Скопируй результат — это будет `FIREBASE_CREDENTIALS`

### Шаг 3 — Задеплой на Railway
1. Зарегистрируйся на [railway.app](https://railway.app) (бесплатно)
2. **New Project → Deploy from GitHub repo** (залей файлы `bot.py` и `requirements.txt`)
3. В разделе **Variables** добавь:
   - `BOT_TOKEN` = токен от BotFather
   - `FIREBASE_CREDENTIALS` = base64 строка из шага 2
4. Railway автоматически запустит бота!

> 💡 **Альтернатива:** Можно просто положить `serviceAccount.json` рядом с `bot.py` и запустить локально:
> ```bash
> pip install -r requirements.txt
> BOT_TOKEN="твой_токен" python bot.py
> ```

### Шаг 4 — Подключи аккаунт
1. Открой сайт с финансами → **Меню → Настройки**
2. Там появится твой **Firebase UID** — нажми "Копировать"
3. Напиши боту: `/uid ВАШ_UID`
4. Готово! 🎉

## 📝 Команды бота

| Команда | Пример | Описание |
|---|---|---|
| Расход | `расход 500 кофе` | Добавить расход |
| Расход (быстро) | `-500 такси` | Быстрый ввод расхода |
| Доход | `доход 50000 зарплата` | Добавить доход |
| Доход (быстро) | `+5000 фриланс` | Быстрый ввод дохода |
| `/баланс` | — | Текущий баланс |
| `/стат` | — | Топ-5 расходов за месяц |
| `/последние` | — | Последние 5 записей |
| `/удалить` | — | Удалить последнюю запись |
| `/uid` | `/uid abc123` | Привязать Firebase аккаунт |
| `/отвязать` | — | Отвязать аккаунт |
| `/помощь` | — | Список команд |

## 🔗 Автоопределение категорий

Бот автоматически определяет категорию по описанию:

- `кофе`, `еда`, `ресторан` → 🍕 Еда
- `такси`, `метро`, `бензин` → 🚗 Транспорт  
- `аптека`, `врач` → 💊 Здоровье
- `кино`, `netflix` → 🎬 Развлечения
- `зарплата` → 💼 Зарплата
- и многое другое!

## 📂 Структура файлов

```
bot.py              — основной код бота
requirements.txt    — зависимости Python
serviceAccount.json — сервисный ключ Firebase (НЕ загружай в git!)
users.json          — связь telegram_id → firebase_uid (создаётся автоматически)
```

> ⚠️ Добавь `serviceAccount.json` и `users.json` в `.gitignore`!
